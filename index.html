Responsive slider with audio support.

A-Slider: Easy-to-use, highly configurable slider

Slide any content, not just images.
Specify (optional) audio for each slide.
Set slide duration individually for each slide.
All configuration in HTML, no Javascript necessary.

Slider Features

  • No external dependencies; works well with Twitter Bootstrap, Foundation, and AngularJs
  • Specify optional audio for each slide
  • Specify slide duration individually for each slide
  • Supports multiple sliders on the same page
  • Uses CSS 3 transitions for better performance
  • Slide any content, not just images
  • All configuration done using HTML attributes - no javascript configuration needed

Installation

  • Install with Bower: 
    bower install a-slider
  • or download as a Zip File
